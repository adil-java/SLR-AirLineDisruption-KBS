## About this project

This folder contains the complete documentation and artefacts from the
Systematic Literature Review (SLR) conducted to support the design of a
Knowledge-Based Expert System (KBS) for Airline Disruption Management and
Recovery Planning. The SLR provides the evidential foundation for the
system's architecture, knowledge representation, algorithmic design, and
identification of research gaps.

## Topic relevance

Airline disruption management (handling Irregular Operations — IROPS) is
one of the most complex real-time decision problems in transportation. It
requires coordinating aircraft, crew, and passengers under tight time
constraints while respecting regulatory limits and operational policies.
Existing systems lack explainability, integration, and passenger-centricity.
A Knowledge-Based Expert System directly addresses these gaps by encoding
domain expertise in an auditable, maintainable form that supports transparent
real-time recovery recommendations.
